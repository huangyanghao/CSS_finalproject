# CSS Final Project

这是我在前端学习阶段的 CSS 最终项目，主要目的是练习 HTML + CSS 页面构建、布局设计与响应式实现。

## 🖼️ 项目预览

![screenshot](./screenshot.png) <!-- 如果你添加了页面截图，可以展示 -->

> 页面包括导航栏、主内容区、响应式布局和底部栏，注重语义结构与美观设计。

---

## 🚀 技术栈

- **HTML5** - 语义化结构
- **CSS3** - 样式与布局控制
  - Flexbox 布局
  - 媒体查询实现响应式
- 💻 可扩展：后期可以接入 JS 动效、或使用 Tailwind CSS 框架升级
